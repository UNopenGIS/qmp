# ブルガリア (Bulgaria) 🇧🇬

## 情報源

- データ収集日: 2025年06月01日
- 主要情報源: OpenStreetMap Wiki, osmstats.neis-one.org, JICA公開情報

## 備考

ブルガリアは東欧・バルカン半島の重要な国家として、スラブ文化と地中海文化が融合した独特の文化を持ちます。EU加盟国として発展を続けており、ITセクターや観光業の成長が著しい国です。

## OSMコミュニティの状況

- **コミュニティ規模**: 中規模（18人/日）
- **活動特性**: ソフィア中心の都市マッピング、黒海沿岸観光地の詳細記録
- **地域特性**: バルカン半島の重要なOSMコミュニティ
- **成長傾向**: 着実な成長、IT関係者・大学生の参加増加

- **第4段階**: 山岳・国境地域の基礎情報整備

## 言語サポート

### 主要言語
- **ブルガリア語** (公用語): `name:bg=*`
- **トルコ語** (少数民族言語): `name:tr=*`
- **ロマ語** (少数民族言語): `name:rmn=*`
- **英語** (国際語): `name:en=*`
- **ロシア語** (歴史的言語): `name:ru=*`

### タグ付け例
```
name=София
name:bg=София
name:en=Sofia
name:tr=Sofya
name:ru=София
old_name=Средец
```
