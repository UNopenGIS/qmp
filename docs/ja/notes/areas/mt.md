# マルタ / Malta

- **正式国名**: マルタ共和国 / Repubblika ta' Malta
- **首都**: バレッタ / Valletta
- **人口**: 約53万人（2023年推定）
- **公用語**: マルタ語、英語
- **主要言語**: マルタ語、英語、イタリア語
- **タイムゾーン**: CET (UTC+1)
- **通貨**: ユーロ (EUR)
- **ISO 3166-1**: MT / MLT / 470
