# ラトビア / Latvia

## 基本情報

- **正式国名**: ラトビア共和国 / Latvijas Republika
- **首都**: リガ / Rīga
- **人口**: 約186万人（2023年推定）
- **公用語**: ラトビア語
- **主要言語**: ラトビア語、ロシア語、リトアニア語
- **タイムゾーン**: EET (UTC+2)
- **通貨**: ユーロ (EUR)
- **ISO 3166-1**: LV / LVA / 428
